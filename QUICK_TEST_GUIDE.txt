================================================================================
  COMMON ISSUES & FIXES
================================================================================

Emulator won't start:
  → Check SDK path is correct
  → Ensure "Pixel6_AOSP33" AVD exists: adb devices -l
  → Kill lingering java/emulator processes: taskkill /F /IM java.exe

APK install fails:
  → Clear cache: adb uninstall com.armutlu.apporganizer
  → Check APK exists: app\build\outputs\apk\debug\app-debug.apk
  → Check storage: adb shell df /data | grep -i bytes

App crashes on launch:
  → Check logcat: adb logcat | grep -i "apporganizer\|error"
  → Dump logs: adb logcat -d > crash.txt
  → Share logs + screenshot with Hüseyin

Logcat shows too much noise:
  → Filter by tag: adb logcat | grep "TAG_NAME"
  → Or use: adb logcat | grep -E "error|crash|exception"
