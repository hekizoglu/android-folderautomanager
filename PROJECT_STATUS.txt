🏗️  ARCHITECTURE DETAYLAR

Database Schema:
   apps (table)
   ├── packageName (PRIMARY KEY)
   ├── appName (TEXT, INDEXED)
   ├── categoryId (FOREIGN KEY → categories)
   ├── iconUrl, isSystemApp, isInstalled
   ├── installTime, lastUpdated, customNotes
   └── CRUD: Insert, Update, Delete, Select

   categories (table)
   ├── categoryId (PRIMARY KEY)
   ├── categoryName (TEXT)
   ├── colorHex, iconEmoji
   ├── isSystemCategory (BOOLEAN)
   ├── displayOrder (INTEGER)
   └── CRUD: Insert, Update, Delete, Select

Data Flow:
   UI Screen
      ↓
   ViewModel (State Management)
      ↓
   Repository (Business Logic)
      ↓
   AppDao / CategoryDao
      ↓
   Room Database (SQLite)
      ↓
   Device Storage

Async Pattern:
   Coroutines + Flow
   └── Non-blocking database operations
   └── Real-time UI updates
   └── Lifecycle-aware subscriptions
